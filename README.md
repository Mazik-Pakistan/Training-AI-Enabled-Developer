## ğŸ› ï¸ Setup Instructions

### 1. Create GitHub Account
1. Go to [github.com](https://github.com)
2. Sign up for a free account
3. Verify your email address

### 2. Get GitHub Copilot Access
**Students:** Free via [GitHub Student Developer Pack](https://education.github.com/)
**Individuals:** $10/month or $100/year
**Organizations:** Business and Enterprise plans available

### 3. Install VS Code
1. Download from [code.visualstudio.com](https://code.visualstudio.com/)
2. Install for your operating system
3. Launch and familiarize yourself with the interface

### 4. Install GitHub Copilot Extensions
1. Open VS Code Extensions (`Ctrl+Shift+X`)
2. Search for "GitHub Copilot"
3. Install both:
   - GitHub Copilot
   - GitHub Copilot Chat
4. Sign in when prompted
