## ğŸ†˜ Common Issues & Fixes

### Issue: Copilot Not Suggesting
**Fix:**
1. Check icon in bottom-right (should be active)
2. Click icon â†’ Sign in if needed
3. Restart VS Code
4. Check internet connection

### Issue: Suggestions Not Relevant
**Fix:**
1. Write clearer comments
2. Add more context
3. Use descriptive variable names
4. Try rewording your prompt

### Issue: Can't Sign In
**Fix:**
1. Try different browser for auth
2. Check GitHub account has Copilot access
3. Clear VS Code cache
4. Reinstall extension
