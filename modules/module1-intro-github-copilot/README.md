# Module 1: Introduction to GitHub & Copilot

## Module Overview
This module introduces you to GitHub platform fundamentals, GitHub Copilot AI assistant, and responsible AI principles. You'll learn how to set up your development environment and start leveraging AI-powered tools for modern software development.

## Units in This Module

### [Unit 1: Introduction to GitHub](./unit1-intro-github.md)
- Overview of GitHub platform
- Core features of GitHub
- How GitHub Codespaces enables faster, frictionless development

### [Unit 2: Introduction to GitHub Copilot](./unit2-intro-copilot.md)
- What GitHub Copilot is and how it works
- Effective prompting for better development experience
- Principles of Responsible AI

### [Unit 3: GitHub Copilot Primer & Environment Setup](./unit3-setup.md)
- GitHub Copilot plans
- GitHub Copilot Chat Modes
- Setting up development environment


## Prerequisites
- Basic understanding of version control concepts
- A GitHub account (free or paid)
- A code editor (VS Code recommended)


## Learning Objectives
By the end of this module, you will be able to:
- ✅ Navigate and use GitHub's core features
- ✅ Explain what GitHub Copilot is and how it works
- ✅ Apply responsible AI principles in development
- ✅ Set up and configure GitHub Copilot in your environment
- ✅ Use GitHub Codespaces for development
