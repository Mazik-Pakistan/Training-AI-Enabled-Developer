# Module 2: Foundations & Inline Code

## Module Overview
This module explores GitHub Copilot's core capabilities for inline code assistance and multi-environment support. You'll learn how to leverage AI-powered code completion, inline code actions, and work seamlessly across multiple programming languages.

## Units in This Module

### Unit 1: Inline Code Assistance
- AI Code Completion in GitHub Copilot
- Inline Code Actions (Explain/Fix/Move/Rewrite)

### Unit 2: Multi-Environment Support
- Work across multiple programming languages
- GitHub Copilot Ask Mode
- Explore different GitHub Copilot interaction modes

## Learning Path
1. Master **Unit 1** for inline code assistance techniques
2. Explore **Unit 2** for multi-language development
3. Complete the **Module Lab** for practical application

## Prerequisites
- Completed Module 1
- GitHub Copilot installed and configured
- Basic programming knowledge in at least one language

## Learning Objectives
By the end of this module, you will be able to:
- ✅ Use AI code completion effectively
- ✅ Apply inline code actions to improve code quality
- ✅ Work with Copilot across multiple programming languages
- ✅ Use different interaction modes appropriately
- ✅ Leverage Copilot's context awareness
