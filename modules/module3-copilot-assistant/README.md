# Module 3: Copilot As Assistant / Conversational Development Workflows

## Module Overview
This module focuses on leveraging GitHub Copilot's chat interface for conversational development. You'll learn advanced prompting techniques, context-aware interactions, and how to have productive conversations with AI to enhance your development workflow.

## Units in This Module

### Unit 1: Copilot Chat Interface
- Chat-based development using Copilot sidebar
- Slash commands (/explain, /fix, etc.)
- Chat Context Anchors (file, selection, terminal)
- Model Selection (GPT-4 / GPT-o1) in GitHub Copilot

### Unit 2: Context-Aware Prompting
- Understanding Context in Prompt Design
- Types of Context Used in Prompts
- Context-Aware Prompting vs Basic Prompting
- Context-Aware Prompting with Zero-Shot, One-Shot, and Few-Shot Learning

## Learning Path
1. Master **Unit 1** for chat interface proficiency
2. Learn **Unit 2** for advanced prompting techniques
3. Apply knowledge in the **Module Lab**

## Prerequisites
- Completed Modules 1 and 2
- GitHub Copilot Chat enabled
- Familiarity with basic Copilot features

## Estimated Time
- **Total Module Time**: 4-5 hours
- Unit 1: 2-2.5 hours
- Unit 2: 1.5-2 hours
- Lab: 1 hour

## Learning Objectives
By the end of this module, you will be able to:
- ✅ Use Copilot Chat effectively for development
- ✅ Master all slash commands
- ✅ Leverage context anchors for better results
- ✅ Write context-aware prompts
- ✅ Apply zero-shot, one-shot, and few-shot learning
- ✅ Select appropriate models for different tasks
