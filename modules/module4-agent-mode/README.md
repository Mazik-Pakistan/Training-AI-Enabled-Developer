# Module 4: Copilot Agent Mode

## Module Overview
Copilot Agent Mode represents the next evolution in AI-assisted development, where Copilot can autonomously handle complex, multi-step tasks. This module teaches you how to leverage agent capabilities to automate workflows, implement features, and boost productivity.

## Unit in This Module

### Unit 1: Get Started with Copilot Agent Mode
- Introduction to Agent Mode
- Writing Custom Instructions for Agents
- Prompt Files in GitHub Copilot
- GitHub Copilot Chat Vs GitHub Copilot Agent

## Learning Path
1. Understand **Agent Mode fundamentals**
2. Practice **writing custom instructions**
3. Create **prompt files** for reusable patterns
4. Complete the **Module Lab** with autonomous tasks

## Prerequisites
- Completed Modules 1, 2, and 3
- Understanding of conversational development
- Experience with complex development tasks

## Estimated Time
- **Total Module Time**: 3-4 hours
- Unit 1: 2-2.5 hours
- Lab: 1-1.5 hours

## Learning Objectives
By the end of this module, you will be able to:
- ✅ Understand when to use Agent Mode vs Chat
- ✅ Write effective custom instructions
- ✅ Create and use prompt files
- ✅ Delegate complex tasks to Copilot Agent
- ✅ Review and validate agent-generated code
- ✅ Integrate agent workflows into your development process
